# 📦 Quantum Store — Inventory Management System

Live inventory management system for Quantum Store.

## 🔗 Live App
Deployed at: `https://YOUR-USERNAME.github.io/quantum-store`

## Features
- Product inventory with stock alerts
- Vendor & purchase order management
- Multi-item dispatch with printable slip
- Delhivery live tracking
- Google Sheets sync
- Admin / User / Employee login roles

## Local Development

```bash
npm install
npm run dev
```

## Deploy
Push to `main` branch → GitHub Actions builds and deploys automatically to GitHub Pages.
